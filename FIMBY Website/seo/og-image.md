# Open Graph share image — action required

## Status: **Not created yet**

AIOSEO and social platforms expect a default **1200×630 px** image for link previews (Facebook, LinkedIn, iMessage, some AI citation cards).

## Recommended asset

| Property | Value |
|----------|-------|
| Dimensions | 1200 × 630 px |
| Format | PNG or JPG |
| Background | FIMBY cream (`#F5F0E8`) or brand gradient |
| Content | FIMBY grass logo + tagline: *Turn the place you live into a place you belong.* |
| Optional subline | *Web · iPhone · Android — our.fimby.com* |
| File name | `fimby-og-share.png` |
| Upload to | WordPress Media Library → copy URL |

## Where to set it

1. **AIOSEO → Social Networks → Facebook → Default Post Image** — paste uploaded URL  
2. **Home page AIOSEO panel → Social tab** — override with same image if needed  

## Interim fallback

Until a custom image exists, AIOSEO may use the site logo or first page image. Acceptable short-term; replace when a designed asset is ready.

## Do not use

- App mockup screenshots (too busy at thumbnail size)  
- Photos with faces without consent for global share card  
- Text smaller than ~24px at full size (illegible in previews)
