# Play tablet exports (1920×2560)

Framed Google Play **10-inch tablet** screenshots. Reuses composited raw captures from `../ipad/`.

## DevTools capture settings (framed finals)

| Setting | Value |
|---|---|
| **Viewport width** | **1920** |
| **Viewport height** | **2560** |
| **Device pixel ratio (DPR)** | **1** |
| **Zoom** | 100% |

Chrome: open an `export-*.html` file → DevTools → **Responsive** → 1920×2560 → DPR **1** → **Capture full size screenshot**.

Save each export to `../../play-tablet/Screen 1.png` … `Screen 8.png`.
