# Play phone exports (1080×1920)

Framed Google Play phone screenshots. Reuses raw captures from `../iphone/` — no new app captures needed.

## DevTools capture settings (framed finals)

| Setting | Value |
|---|---|
| **Viewport width** | **1080** |
| **Viewport height** | **1920** |
| **Device pixel ratio (DPR)** | **1** |
| **Zoom** | 100% |

Chrome: open an `export-*.html` file → DevTools → toggle device toolbar → **Responsive** → enter 1080×1920 → set DPR to **1** in the device menu (or `chrome://flags` off if needed) → **Capture full size screenshot**.

Save each export to `../../play-phone/Screen 1.png` … `Screen 8.png`.
